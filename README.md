# Track A: Casimir/Lattice QED

## Overview
Track A explores Casimir/Lattice Quantum Electrodynamics (QED), focusing on vacuum energy extraction through the optimization of material geometries. This track models the physical forces generated by vacuum fluctuations at the nanoscale.

## Current Simulation: Casimir Pressure
The primary application, `casimir_sim.py`, calculates the attractive pressure between parallel plates. This provides the baseline force metric required for lattice optimization.

### Baseline Performance (Ideal Plates):
- **10nm:** -1.30e+05 Pa (~1.28 atm)
- **100nm:** -1.30e+01 Pa
